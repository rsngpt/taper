AgentFailure

data class AgentFailure(val httpStatus: Int? = null, val exception: Throwable? = null, val responseBody: String? = null)

Everything the classifier may inspect about a failure. All fields optional: a pure transport failure has only exception; an HTTP-level failure has httpStatus and usually responseBody.

Parameters

responseBody

the error response body, if any. Callers should cap what they retain here (error bodies are small; do not hand the classifier a multi-megabyte success payload).
